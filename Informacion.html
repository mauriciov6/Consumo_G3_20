Libro

El formulario Libro sirve para llevar el control de los libros existentes en la base de datos G3_20 creada en MySQL. Por cada libro llevamos el registro de los siguientes datos: El código de libro, nombre del libro, escritor del libro, su fecha de publicación, ISBN, precio y la editorial. Con este formulario podemos listar, actualizar, eliminar y agregar nuevos.

Escritor

El formulario Escritor sirve para recopilar información de manera estructurada, para después almacenarla en nuestro sistema de base de datos y está conformado por siete campos en específico, los datos a solicitar son de acuerdo al escritor que se quiere registrar o en dado caso se requiera hacer la actualización de datos recopilados y está también la opción de poder eliminar el escritor de nuestro sistema.

Editorial

El formulario Editorial realiza el trabajo para llevar el control de lo que es número de editorial, nombre editorial, dirección, país, fecha de fundación, cantidad de libros impresos y numeros de teléfono existentes en la base de datos G3_20 creada en MySQL. Con este formulario podemos registrar, actualizar, eliminar y agregar nuevos registros.

Alquiler

El formulario Alquiler sirve para llevar el control de los alquileres existentes en la base de datos G3_20 creada en MySQL. Por cada alquiler llevamos el registro de los siguientes datos: El código de libro, nombre del libro, fecha de alquiler, nombre del cliente, direccion, dias a alquilar y precio a alquilar. Con este formulario podemos listar, actualizar, eliminar y agregar nuevos.
